# 📈 Stock Portfolio Tracker

## 📌 Description
A simple Python-based Stock Portfolio Tracker that calculates the total investment value based on predefined stock prices and user-entered quantities.

## ✨ Features
- User-friendly console interface
- Calculates investment value
- Supports multiple predefined stocks
- Displays total portfolio value
- Simple and beginner-friendly

## 🛠 Technologies Used
- Python 3

## ▶️ How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

```bash
python portfolio.py
```

## 📷 Sample Output

```
📈 Stock Portfolio Tracker

Enter stock name: AAPL
Enter quantity: 5

Total Investment Value: $900
```
